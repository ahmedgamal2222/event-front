## 🎯 أين يظهر كل شعار؟

### 1️⃣ **شعار الحدث (Event Logo)** 🏢

```
❌ الصفحة الرئيسية   → لا يظهر للزائرين
❌ الـ Navbar        → لا يظهر
❌ الـ Footer        → لا يظهر

✅ نموذج التسجيل    → يظهر في الأعلى
✅ البريد الإلكتروني → يرسل معك
✅ الإشعارات        → يظهر بجانب الرسالة
✅ التطبيقات        → يظهر في الواجهات
```

### 2️⃣ **صورة الغلاف (Cover Image)** 📸

```
❌ الـ Navbar        → لا
❌ الـ Footer        → لا
❌ نموذج التسجيل    → لا

✅ Hero Section      → يملأ كل الشاشة الأولى
✅ خلفية الحدث      → تظهر في الصفحة
```

### 3️⃣ **شعار الصفحة (Page Logo)** 🎪

```
❌ نموذج التسجيل     → لا يظهر
❌ البريد            → لا يظهر

✅ الـ Navbar        → إذا اخترت "navbar" أو "both"
✅ الـ Footer        → إذا اخترت "footer" أو "both"
✅ الزائرين فقط     → يراها الزائرون فقط
```

## ⚡ ملخص سريع

### **متى استخدم أي شعار؟**

| الموقف | استخدم |
|--------|--------|
| أريد شعار في نموذج التسجيل | Event Logo (شعار الحدث) |
| أريد صورة خلفية جميلة | Cover Image (صورة الغلاف) |
| أريد شعار في أعلى الصفحة | Page Logo (شعار الصفحة) + Navbar |
| أريد شعار في أسفل الصفحة | Page Logo (شعار الصفحة) + Footer |
| أريد شعار في الاثنين | Page Logo + Both |

### **ماذا لو لم أرفع صورة؟**
```
❌ Event Logo    → لن تظهر أيقونة في النموذج
❌ Cover Image   → سيظهر لون عادي فقط
❌ Page Logo     → لن يظهر شعار في الـ navbar/footer
```

### **هل يمكن تغيير الصور بعد الحفظ؟**
```
✅ نعم! في أي وقت
✅ اضغط تعديل → اختر صورة جديدة
✅ الصورة القديمة تُحذف تلقائياً
```

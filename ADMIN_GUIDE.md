# 📖 دليل الإدارة الشامل

## 🎯 مقدمة
هذا الدليل يشرح كل ميزات لوحة الإدارة بالتفصيل.

## 📊 الأقسام الرئيسية

### 1️⃣ **معلومات الحدث** ⚙️
المكان الرئيسي لإدارة بيانات الحدث الأساسية.

#### **البيانات الأساسية:**
| الحقل | الشرح | مثال |
|--------|--------|--------|
| اسم الحدث (EN/AR) | اسم الحدث | S3 Summit / قمة الشركات الناشئة |
| الشعار | جملة توضيحية | "Syria Startups Summit" |
| الوصف | وصف الحدث | "ثلاثة أيام من الإلهام والابتكار..." |
| الموقع | مكان الحدث | دمشق، سوريا |
| التواريخ | متى الحدث | من 2026-12-25 إلى 2026-12-27 |
| البريد | للتواصل | info@event.com |
| الروابط الاجتماعية | تويتر، إنستغرام، لينكدإن | @eventname |
| اللون الرئيسي | لون الموقع | #6C63FF |

#### **الشعارات الثلاثة:**
```
🏢 شعار الحدث (Event Logo)
├─ المكان: في "معلومات الحدث"
├─ الظهور: في النماذج، البريد الإلكتروني، الإشعارات
├─ الاستخدام: شعار رسمي للحدث
└─ مثال: شعار الشركة/الحدث الأسود

📸 صورة الغلاف (Cover Image)
├─ المكان: في "معلومات الحدث"
├─ الظهور: خلفية الحدث في التطبيق
├─ الاستخدام: صورة رئيسية جميلة
└─ الحد الأقصى: 4MB

🎨 شعار الصفحة (Page Logo)
├─ المكان: في "محتوى الصفحة"
├─ الظهور: في الـ Navbar و/أو Footer
├─ الاستخدام: شعار للزائرين
├─ الموضع: أعلى / أسفل / الاثنين
└─ الحد الأقصى: 3MB
```

#### **حالة الحدث:**
- **مسودة**: لم ينشر بعد
- **منشور**: متاح للزوار والتسجيل
- **مؤرشف**: انتهى الحدث

### 2️⃣ **التسجيلات** 📋
إدارة التسجيلات والمشاركين.

#### **ما هي التسجيلات؟**
التسجيلات هي الأشخاص الذين ملأوا نموذج التسجيل للحضور أو المشاركة.

#### **أنواع التسجيلات:**
```
🚀 شركة ناشئة          → شركات ناشئة تريد العرض
💼 مستثمر              → مستثمرون في البحث عن فرص
👤 حضور عام            → أشخاص يريدون الحضور فقط
🎙️ متحدث               → متحدثون في الحدث
🏅 راعي                → شركات راعية
📹 إعلام               → وسائل إعلام
```

#### **حالات التسجيل:**
```
🟡 قيد الانتظار (Pending)    → جديد، لم يتم الفحص
✅ مقبول (Approved)         → تم الموافقة، يمكنه الحضور
❌ مرفوض (Rejected)         → تم الرفض
⏳ في قائمة الانتظار        → في الانتظار (إذا امتلأ الحدث)
✕ ملغى (Cancelled)         → إلغى التسجيل
```

#### **الفلاتر والبحث:**
```
🔍 البحث      → ابحث بالاسم أو البريد الإلكتروني
📊 الحالة      → فلتر حسب حالة التسجيل
🏷️ النوع      → فلتر حسب نوع التسجيل (الأنواع المفعلة فقط)
📄 الصفحات   → تصفح التسجيلات (20 لكل صفحة)
```

#### **معلومات كل تسجيل:**
| الحقل | الشرح |
|--------|--------|
| الاسم | اسم المسجل |
| البريد | بريده الإلكتروني |
| النوع | نوع التسجيل (ملون) |
| الوصف | مجال عمله أو سبب مشاركته |
| المدينة | مدينته |
| الحالة | الحالة الحالية |
| التاريخ | متى سجل |
| تغيير الحالة | اختر حالة جديدة |

### 3️⃣ **المتحدثون** 🎙️
إدارة المتحدثين والعاملين بالحدث.

#### **إضافة متحدث:**
```
الاسم (EN/AR)       → اسم المتحدث
المسمى               → وظيفته (مثل: المؤسس، الرئيس)
الشركة/المنظمة    → مكان عمله
الصورة              → صورة شخصية
النبذة              → نبذة عن المتحدث
مميز                → هل متحدث رئيسي؟
مفاجأة              → متحدث مفاجأة (لم يعلن عنه بعد)
```

#### **حذف/تعديل الصور:**
- عند رفع صورة جديدة ، الصورة القديمة تُحذف تلقائياً
- اضغط على الصورة → زر الحذف ✕ لحذفها فوراً

---

### 4️⃣ **البرنامج** 📅
إدارة جدول الحدث الزمني والجلسات.

#### **الخطوة 1: إضافة أيام**
```
1. اضغط "+ يوم جديد"
2. أدخل:
   - رقم اليوم      → 1, 2, 3 ...
   - التاريخ        → مثال: 2026-12-25
   - التسمية (AR)   → اليوم الأول
   - التسمية (EN)   → Day One
3. اضغط "حفظ"
```

#### **الخطوة 2: إضافة جلسات**
```
تحت كل يوم:
1. اضغط "+ جلسة جديدة"
2. أدخل:
   - الوقت الابتداء   → 09:00
   - الوقت الانتهاء   → 10:00
   - العنوان (AR)     → اسم الجلسة
   - نوع الجلسة:
     * محاضرة (talk)
     * ورشة عمل (workshop)
     * حوار (panel)
     * فترة توعية (keynote)
     * تواصل (networking)
     * فترة راحة (break)
     * مسابقة (competition)
   - المتحدث (اختياري)
   - الترتيب           → الترتيب في اليوم
   - الوصف (AR)        → وصف الجلسة
3. اضغط "حفظ"
```

#### **تعديل التواريخ:**
```
لتغيير تاريخ يوم:
1. اضغط على اليوم المراد تعديل تاريخه
2. غير حقل "التاريخ"
3. اضغط "حفظ"

لنقل جلسة من يوم لآخر:
1. اضغط "تعديل" على الجلسة
2. غير "اليوم" من القائمة المنسدلة
3. اضغط "حفظ"
```

### 5️⃣ **الرعاة** 🏅
إدارة الشركات الراعية والشركاء.

#### **إضافة راعي:**
```
الاسم               → اسم الشركة
الشعار              → شعار الشركة (صورة)
الموقع الإلكتروني  → رابط الشركة
مستوى الرعاية      → بلاتيني / ذهبي / فضي / برونزي / إعلامي
الترتيب            → ترتيب العرض
```

#### **مستويات الرعاية:**
- **بلاتيني (Platinum)**: أعلى مستوى، أكثر ظهور
- **ذهبي (Gold)**: ظهور كبير
- **فضي (Silver)**: ظهور عادي
- **برونزي (Bronze)**: ظهور محدود
- **إعلامي (Media)**: وسائل إعلام

---

### 6️⃣ **الأسئلة الشائعة** ❓
إدارة الأسئلة والأجوبة.

#### **إضافة سؤال:**
```
السؤال (AR)         → مثال: "متى ينتهي التسجيل؟"
الجواب (AR)         → الإجابة الكاملة
الترتيب             → ترتيب الظهور
```

---

### 7️⃣ **نموذج التسجيل** 📝
تخصيص نموذج التسجيل وحقوله.

#### **الخيارات الرئيسية:**
```
أنواع التسجيل المفعلة    → أي أنواع تريد التسجيل لها؟
حالة التسجيل           → هل النموذج مفتوح؟
عنوان النموذج          → "سجل في القمة"
الحقول المطلوبة        → الهاتف، المدينة، إلخ
المدن المتاحة          → اختر المدن
القطاعات              → مجالات العمل
مراحل الشركة          → من فكرة لنمو
```

### 8️⃣ **محتوى الصفحة الرئيسية** 🎨

#### **قسم Hero (الأول):**
```
🏆 الاختصار         → S3, EF, إلخ
🎯 الزر الرئيسي     → "سجل الآن" أو "تفاصيل أكثر"
🔘 الزر الثانوي     → اتركه فارغاً للإخفاء
```

#### **الإحصائيات:**
```
كل إحصائية:
- التسمية         → "أيام من الإلهام"
- مصدر البيانات   → days_count, startup_count, إلخ
- القيمة الافتراضية → إذا لم تتوفر البيانات
```

#### **عن الحدث (البطاقات):**
```
كل بطاقة:
- الرمز (Emoji)   → 🚀, 🤝, 💡, إلخ
- العنوان         → "إطلاق الأفكار"
- الوصف           → شرح مختصر
```

#### **الشعار (Page Logo):**
```
رابط الشعار        → صورة أو رابط
مكان الشعار        → 
  • 🔝 Navbar فقط    → يظهر أعلى الصفحة
  • 🔙 Footer فقط    → يظهر أسفل الصفحة
  • ↕️ الاثنين        → يظهر أعلى وأسفل
```

## 💡 نصائح مهمة

### **حول الصور:**
1. **R2 Upload**: جميع الصور ترفع على Cloudflare R2 (سريع وآمن)
2. **تغيير الصورة**: عند رفع صورة جديدة، القديمة تُحذف تلقائياً
3. **حذف الصورة**: اضغط الزر "✕ حذف" بجانب الصورة
4. **المعاينة**: شاهد الصورة قبل الحفظ

### **حول التسجيلات:**
1. **الفلترة**: استخدم الفلاتر للعثور على تسجيلات محددة
2. **الحالات**: غير الحالة حسب قرارك (موافقة/رفض/انتظار)
3. **البيانات الإضافية**: شاهد تفاصيل كاملة في البطاقة الجانبية

### **حول البرنامج:**
1. **التواريخ**: تأكد من كل التواريخ صحيحة
2. **الوقت**: تأكد من أوقات الجلسات متسلسلة
3. **النقل**: يمكنك نقل جلسة من يوم لآخر بسهولة

### **حول الصفحة:**
1. **الإحصائيات**: تحدث تلقائياً من البيانات
2. **الشعار**: لا تنسى اختيار موضع الشعار
3. **الألوان**: اختر لون يطابق هوية الحدث

```
✅ يمكنك التراجع عن التغييرات بتعديل ثاني
❌ تأكد قبل حذف المتحدثين أو الجلسات

🔐 بيانات حساسة:
- كل بريد المسجلين محفوظ
- لا يمكن نشر البيانات
- استخدم البيانات للتواصل فقط
```

## ❓ الأسئلة الشائعة

**س: هل يمكنني تعديل حقل بعد الإضافة؟**
ج: نعم، اضغط على العنصر ثم "تعديل"

**س: ماذا لو حذفت متحدث عن طريق الخطأ؟**
ج: يمكنك إضافته مرة أخرى

**س: هل التسجيلات تُحفظ تلقائياً؟**
ج: نعم، كل تغيير يُحفظ فوراً

**س: كم حد أقصى للصور؟**
ج: 
- صورة غلاف: 4MB
- شعار: 3MB

**س: هل يمكن إخفاء قسم من الصفحة؟**
ج: نعم، اترك الحقول فارغة
